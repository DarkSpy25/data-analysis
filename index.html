Analysis of sample dataset

(Cars-Engage-2022)
  • The dataset given contains only vehicle specifications and features.
  • The lack of sales data makes it very tough to make smart predictions.
  • The columns towards the end of the dataset contain a lot of missing values.
  • The blank cells indicate absence of information. This makes it hard to draw charts.
  • Some of the columns contain extra text(eg. 1024 kg kg), some contain wrong values(eg. 1025#! kg) and some like the emission norms column have no consistency(BS6 and BSVI)
  • Therefore, a lot of steps were taken to clean the data. Isolates the most important columns to make smart predictions. Cleaned that data.
  • Removed unnecessary columns. Also removed extra text and converted to numeric values. Unmerging of cells was also part of the data clean up process.
